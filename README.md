# Customer Churn Analysis

## Project Overview
This project performs end-to-end customer churn analysis using **SQL for data cleaning and transformation** and **Power BI for interactive dashboard visualization**.

## Objectives
- Clean and prepare raw customer data using SQL  
- Analyze churn patterns across age, tenure, contract type, services, and demographics  
- Build an interactive Power BI dashboard to generate business insights  

## Tools & Technologies
- SQL (Data Cleaning & Transformation)  
- Power BI (Data Visualization & Dashboard)  
- GitHub (Project Sharing)

## Key Insights
- Higher churn observed in **monthly contract customers**  
- **Fiber optic users** show increased churn rate  
- Churn varies across **age groups and tenure duration**
